## 🚀 Usage

## API

```
const fileRoulette = require('./App')
fileRoulette('./test/file/lol.js')
```

## Quick use
```
$ npx file-roulette -p <path>
```
## CLI Installation
```
$ npm install -g file-roulette
```
```
$ file-roulette -p <path>
```

`` -p <path> `` is optional
